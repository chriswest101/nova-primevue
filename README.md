## Laravel Nova Field With PrimeVue

为避免重复加载，Laravel Nova 项目中，只要用到 PrimeVue 组件的，都请先安装此包。

### Requirements

- `php: ^7.3|^8.0`
- `laravel/nova: ^4.0`

### Supported Components

- Listbox Field

### Installation

Install the package in a Laravel Nova project via Composer:

```bash
composer require jshxl/primevue
```
